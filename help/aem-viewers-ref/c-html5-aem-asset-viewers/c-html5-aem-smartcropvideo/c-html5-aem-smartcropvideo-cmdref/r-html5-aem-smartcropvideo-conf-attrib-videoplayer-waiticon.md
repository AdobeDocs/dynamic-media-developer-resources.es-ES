# SmartCropVideoPlayer.waiticon{#smartcropvideoplayer-waiticon}

Atributo de configuración para el visor de vídeo de recorte inteligente.

`[SmartCropVideoPlayer.|<containerId>_videoPlayer.]waiticon=0|1`

<table id="table_C616483932C2482CA9794DDD7313FD7C"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0 | 1</span> </p> </td> 
   <td colname="col2"> <p> Activa o desactiva la visualización de animación de almacenamiento en búfer (icono de espera). </p> </td> 
  </tr> 
 </tbody> 
</table>

## Propiedades {#section-f42369774e2740dcb399626a0e4e930e}

Opcional.

## Predeterminado {#section-d016470e92a74f98a18c4ab3489410a5}

`1`

## Ejemplo {#section-7621c8ebd4144bc08a537d01bd9c3f2f}

```
waiticon=0
```
