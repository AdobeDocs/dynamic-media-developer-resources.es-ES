# ZoomView.reset{#zoomview-reset}

 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0 | 1</span> </p> </td> 
   <td colname="col2"> <p> Restaura el puerto de vista cuando cambia el marco (imagen). Si está configurado como <span class="varname"> 0</span>, conserva el puerto de vista actual con el mejor ajuste posible, preservando al mismo tiempo la relación de aspecto de la imagen recién establecida. </p> </td> 
  </tr> 
 </tbody>
