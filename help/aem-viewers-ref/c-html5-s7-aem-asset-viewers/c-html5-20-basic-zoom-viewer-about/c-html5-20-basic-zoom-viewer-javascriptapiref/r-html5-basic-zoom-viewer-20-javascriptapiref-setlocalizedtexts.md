Referencia de la API de JavaScript para el visor de zoom básico.

` setLocalizedTexts( *`localizationInfo`*)`

 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> <span class="varname"> localizationInfo</span> </span> </p> </td>
