# imageRes{#imageres}

Define la resolución de los mapas de bits incrustados en una solicitud de impresión virtual.

` imageRes=< *`número`*>`

Se recomienda 300 al imprimir. Cuanto mayor sea el número, más tiempo necesitará para procesar la imagen.
