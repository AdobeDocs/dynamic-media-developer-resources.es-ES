# markOverflowingTextFrames{#markoverflowingtextframes}

Mostrar marcos de texto desbordados con signo más. El indicador de desbordamiento de texto muestra si el texto se sale del espacio que tiene asignado en un marco de texto (o en el último marco de texto si se trata de un texto enlazado). Este indicador es un cuadro rojo con un signo más en su interior.

<table id="simpletable_F17FD29EB52043BF9000923ED5195A26"> 
 <tr class="strow"> 
  <td class="stentry"> <p><span class="codeph"> &amp;markOverflowingTextFrames</span> </p> </td> 
  <td class="stentry"> <p>0|1 </p></td> 
 </tr> 
</table>

Modificador de configuración `markOverflowingTextFrames=1` mediante una llamada URL marca todos los marcos de texto en los que el texto está desbordado por un signo más. Además, en el visualizador previo de Dynamic Media Classic, el indicador de desbordamiento de texto se establece en &quot; `TRUE`&quot; de forma predeterminada.

El valor predeterminado es 0.
