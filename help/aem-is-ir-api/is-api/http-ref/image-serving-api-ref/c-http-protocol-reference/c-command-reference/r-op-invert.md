# op_invert{#op-invert}

`op_invert=0|1`

## Propiedades {#section-2e7a7b340dd147db9bd68ff3bfb6f4c1}

Capa, comando. Se aplica a la capa actual o a la imagen compuesta si `layer=comp`. Ignorado por las capas de efecto.
