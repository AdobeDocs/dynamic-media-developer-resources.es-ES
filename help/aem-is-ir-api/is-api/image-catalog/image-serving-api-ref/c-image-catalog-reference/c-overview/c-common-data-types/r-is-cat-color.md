# Color{#color}

Tipo de datos de color.

Valor de color. Valor RGB hexadecimal y empaquetado, opcionalmente precedido por `0x`. Por ejemplo, el valor RGB `128,255,0` puede especificarse como `0x80ff00` o `80ff00`.
