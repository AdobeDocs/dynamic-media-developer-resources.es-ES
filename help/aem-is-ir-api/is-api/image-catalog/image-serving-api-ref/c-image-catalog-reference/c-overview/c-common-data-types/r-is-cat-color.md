# Color{#color}

Tipo de datos de color.

Valor de color. Valor RGB hexadecimal empaquetado, opcionalmente precedido de `0x`. Por ejemplo, el valor RGB `128,255,0` se puede especificar como `0x80ff00` o `80ff00` .
