# Color{#color}

Tipo de datos de color.

Valor de color. Valor RGB hexadecimal empaquetado, precedido opcionalmente por `0x`. Por ejemplo, el valor de RGB `128,255,0` se puede especificar como `0x80ff00` o `80ff00`
