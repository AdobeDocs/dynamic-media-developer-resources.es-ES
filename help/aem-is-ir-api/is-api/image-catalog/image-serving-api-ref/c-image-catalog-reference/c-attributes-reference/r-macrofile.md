# MacroFile{#macrofile}

Ruta del archivo de definiciones de macros. Especifica el archivo que contiene las definiciones de macros para este catálogo de imágenes.

## Propiedades {#section-e55829aecc3a412f9121a73d70f83a1a}

Valor de cadena de texto. Ruta de archivo absoluta o ruta relativa a la carpeta del catálogo.

## Predeterminado {#section-903de40cff734254aaf64c14258dc4a7}

Vacío, lo que indica que este catálogo de imágenes no tiene definiciones de macros privadas.
