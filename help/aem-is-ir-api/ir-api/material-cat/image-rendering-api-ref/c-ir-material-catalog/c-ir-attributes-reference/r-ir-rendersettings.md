# RenderSettings{#rendersettings}

Configuración de procesamiento avanzada predeterminada. Valor predeterminado para catálogo::RenderSettings.

## Propiedades {#section-fa89db09aeeb420292cf783f07b8547a}

Cadena de texto. Si se especifica, debe ser una cadena de configuración de procesamiento válida, tal como se define con la herramienta de creación de viñetas (que forma parte del paquete de creación de imágenes de Dynamic Media).

## Predeterminado {#section-4e76784aeecd4f09a33124aded7406cb}

`default::RenderSettings` si no se especifica o si está vacío.
