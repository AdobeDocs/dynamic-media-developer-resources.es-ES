# Notas de la versión de Scene7 Publishing System 6.0{#scene-publishing-system-release-notes}

Notas de la versión que describen las funciones, mejoras y correcciones de Adobe Scene7 Publishing System 6.0, aplicación que forma parte de Adobe Marketing Cloud.

## Nuevas funciones y mejoras {#section-4bbd330df5a043f7a56eee619ef7c442}

**General**

* Compatibilidad lingüística con chino.
* Mejoras administrativas, incluidas:

   * Exportar lista de usuarios de la empresa.
   * Enviar correos electrónicos CCO a los usuarios.
   * Restablezca las contraseñas de los usuarios a una contraseña aleatoria que el usuario pueda restablecer.

* Mejora de la facilidad de uso de la vista previa de plantillas. Consulte lo siguiente para obtener más información:

**Visores**

   La función [!DNL Embed Code] está disponible al abrir la página [!DNL Preview: Viewer List] o el panel [!DNL URLs] en [!DNL Detail View] para cualquiera de los siguientes tipos de activos:

   * Visores universales
   * Zoom
   * Catálogos electrónicos
   * Conjuntos de imágenes
   * Conjuntos de muestras
   * Spin
   * Reediciones de vídeo
   * Conjuntos de vídeos adaptables
   * Vídeo
   * Conjuntos de medios mixtos

* Esquemas de metadatos

   Los esquemas de metadatos definen conjuntos de propiedades de metadatos. Cada propiedad del esquema de metadatos tiene un tipo de datos básico como Cadena, Flotante, Entero, Fecha o Sí/No. También se pueden definir listas, opciones abiertas u opciones cerradas. Los esquemas de metadatos se pueden utilizar en todos los flujos de trabajo de metadatos, como búsqueda, edición, importación, vistas de metadatos y ajustes preestablecidos de metadatos. <!-- [More information](http://help.adobe.com/en_US/scene7/using/WS259993e42159a215-1c6a66df1265272619e-7fec.html#WSd968ca97bf03cf72-5e3dd3a113268dc80f5-8000). -->

   Los esquemas de metadatos son la tecnología sucesora de los campos definidos por el usuario y están pensados para flujos de trabajo de metadatos personalizados, especialmente en el contexto del servidor de metadatos.

* Servidor de metadatos

   >[!NOTE]
   >
   >El servidor de metadatos estará disponible a partir de mayo. Debe ser habilitado por el servicio de asistencia. Actualmente, el servidor de metadatos solo está disponible en Norteamérica y Europa.

* El servidor de metadatos expone un subconjunto de todos los campos de metadatos de un recurso en un servidor público y se puede solicitar aquí por medio de servidores o páginas eCom. Estos metadatos expuestos podrían ser un conjunto mixto de propiedades de recursos, campos definidos por el usuario, propiedades de esquema de XMP y metadatos. El servidor de metadatos ofrece una API WEB con una variedad de funcionalidades de búsqueda, como búsquedas de metadatos, similitud de imágenes o ambas. Para cada recurso devuelto se solicita una lista de propiedades de metadatos. <!-- [More information](http://help.adobe.com/en_US/scene7/using/WS1ffbb36e209a6fc9-44279087131d3ad5622-8000.html). -->
* La importación de metadatos puede gestionar propiedades XMP

   La importación de metadatos se amplía para gestionar XMP propiedades y propiedades, definidas en esquemas de metadatos.
* Vista de metadatos para mostrar solo los campos que contengan valores

   Hay una nueva vista de metadatos predefinida denominada &quot;Todas las propiedades con valores&quot; que solo muestra metapropiedades que contienen valores. Las propiedades (de XMP, campos definidos por el usuario o los nuevos esquemas de metadatos) sin valores están ocultas y no aparecerán.
* La configuración de metadatos ahora se agrupa en la interfaz de usuario de administración.

   En **[!UICONTROL Configuración > Configuración de la aplicación]**, las páginas de configuración relacionadas con los metadatos ahora se agrupan en una sección de nivel superior llamada [!DNL Metadata]. Aquí puede encontrar las páginas de configuración para la vista de metadatos, los ajustes preestablecidos de metadatos, el esquema de metadatos y los campos definidos por el usuario.

Esta aplicación de escritorio permite importar modelos 3D existentes y crearlos como archivos de viñeta 2D. Los archivos de viñeta resultantes se pueden cargar en SPS, donde los colores y las texturas se pueden reemplazar bajo demanda.

## Correcciones de software {#section-4219d5dd0a024cd3a95aaac45598363c}

Se han corregido los siguientes problemas en el servidor de metadatos:

* Se ha corregido un problema que provocaba que fallaran las cargas de archivos EPS muy grandes.
* Las &quot;búsquedas de metadatos&quot; con una lista de resultados larga fallan de forma silenciosa. Este problema hace que se muestren resultados de búsqueda incorrectos.
