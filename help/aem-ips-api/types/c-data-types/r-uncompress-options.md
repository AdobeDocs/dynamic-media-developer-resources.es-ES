# DescompressOptions{#uncompressoptions}

Ajuste de carga para procesar archivos ZIP y TAR como recursos principales (Ninguno) o para extraer y cargar su contenido (Descomprimir).

>[!NOTE]
>
>`None` es el valor predeterminado.
