## Parámetros {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| Nombre | Tipo | Descripción |
|---|---|---|

| ` *`destPath`*` | `xsd:string` | Ruta de acceso al archivo IPS y nombre del archivo al que desea cargar. |
